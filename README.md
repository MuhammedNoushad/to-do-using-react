# React Todo App

This is a simple Todo application built using React. It allows users to add, remove, and mark tasks as completed.

## Features

- Add new tasks to the list.
- Mark tasks as completed.
- Remove tasks from the list.
- Edit task.

## Technologies Used

- React
- JavaScript (ES6+)
- HTML
- CSS
